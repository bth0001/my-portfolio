Portfolio

Postradamus

Postradamus is an app where users can make predictions about the future. Users can see others' predictions and agree or disagree with those predictions.

This app was built with a Ruby on Rails back-end acting as an API to serve JSON data to an Angular front-end. Other technologies used for this portfolio include Devise, Omniauth, Bower, and Bootstrap.

Github

Tweetagotchi

Tweetagotchi is a virtual pet that you take care of by tweeting. I built it about six weeks into the General Assembly Web Development Immersive course as my first portfolio. Once a day, the app checks the timestamp of your most recent tweet and if the timestamp is within twenty-four hours, your pet will be fed - otherwise it will grow hungry.

The technologies used for this portfolio include Ruby on Rails, the Twitter API, and Devise.

Github

CrowdSurfer

CrowdSurfer is an app that helps up-and-coming DJ's plan tours by returning danceclub venues in cities where their sound is popular. It was built over a 24-hour period during General Assembly's first hackathon. My main contributions to this portfolio were getting data from the Yelp API and updaing the DOM with JavaScript.

The technologies used for this portfolio include Ruby on Rails, the Yelp API, JavaScript, and Bootstrap.

Github

EgoAmigo

EgoAmigo is an app that allows users to send a random (code-related) compliment to another person via the Twillio API. The Twillio API allows developers to send messages to one designated number for free, and upgrade to a paid plan for additional numbers. So for now this app only works for sending me compliments. My favorite compliment: "Your code is so DRY it makes me thirsty."

The technologies used for this portfolio include Ruby on Rails, the Twillio API, and Heroku.
